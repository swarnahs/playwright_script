# Playwright Product Table Scraper

## Setup

1. Create/activate a Python virtual environment.
2. Install requirements:

   pip install -r requirements.txt
   playwright install

3. Edit `scraper.py` to set your login URL, credentials, and selectors.
4. Run:

   python scraper.py

## Output

- `product_table.json`: Structured table data for analysis.
- `playwright_session.json`: Saved session for re-use.

## Requirements

- Python 3.8+
- VS Code with Python extension (recommended)
